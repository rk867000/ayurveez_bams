# Security Policy

## Reporting Security Issues
If you discover a security vulnerability in this project, please report it responsibly:
- Email: ayurveez@gmail.com
- Do NOT create public GitHub issues for security vulnerabilities

## Security Practices
1. **API Keys**: Never exposed in client-side code
2. **Payments**: Processed through secure UPI/bank channels
3. **Data**: No sensitive user data collected or stored
4. **Updates**: Regular security updates applied

## User Data Protection
- No personal medical data collected
- Payment information handled by secure banking channels
- Contact information used only for course access

## Disclaimer
This is an educational platform. Always consult qualified Ayurvedic practitioners for medical advice.
